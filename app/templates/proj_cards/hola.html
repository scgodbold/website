Holacracy Mailing Lists Zappos
A project kicked off due to the infrastructure change at Zappos. This project will live read the corporate structure and generate postfix mailing lists for the company. These lists were restricted based on the user's position in the company. It also worked with LDAP to allow users to query for mailing addresses inside their mail client of choice.
Technologies Used
  • Python
  • Postfix
  • LDAP / slapd
  • MySQL
