Incident Chat Bot Zappos
A fault resistant jabber bot that notifies, monitors, and documents progress of incidents as they occur in the Zappos environment. The bot also emails interested parties frequent updates to keep the company communication up and reduce downtime.
Technologies Used
  • Python (Flask / Celery)
  • Jabber
  • Amazon Web Services (EC2 / ELB)
  • RabbitMQ
  • Jira API
